1. 클래스 = Class
    클래스는 객체 생성하기 위한 '틀' 또는 설계도 이다.
    클래스는 객체가 가져야 할 '속성'과 '기능'을 정의한다.
    ex) 테슬라 모델 y를 만들어내는 설계도 와 공정 (테슬라모델Y 클래스)

2. 객체 = object
    객체는 클래스에서 정의한 속성과 기능을 가진 실체이다. 객체는 서로 독립적인 상태 가진다.
    ex) 테슬라 모델 Y 설계도와 공정을 이용해서 만들어낸 테슬라 모델 Y
        같은 설계도와 공정을 이용하여 어러대의 테슬라를 뽑아낼수있음. (테슬라모델Y1객체, 테슬라모델Y2객체 .... )

3. 인스턴스 = instance
    인스턴스는 특정 클래스로부터 생성된 객체를 의미한다. 그래서 객체와 인스턴스라는 용어는 자주 혼용된다.
    인스턴스는 주로 어떤 클래스에 속해 있는지 강조할때 사용된다.
    ex) 테슬라모델Y1은 테슬라모델Y를 생산하는 설계도와 공정을 이용해서 만들어낸 차량이다. (테슬라모델Y1인스턴스는 ..~ 이다.)

4. 객체 vs 인스턴스
    둘다 같은 표현이지만. 객체는 클래스를 통해 생성된 어느 하나의 요소를 말하는것에 가깝고
    인스턴스는 이 요소가 어느 클래스를 통해 생성되었다 라는 관계를 명확히 할 때 사용한다.
    즉, 테슬라 모델 Y1는 테슬라 모델Y 설계도와 공정을 통해 생성된 인스턴스 이다. 라고 표현을 한다.
    (사실 그냥 혼용하여도 아무런 문제가 없다.)
